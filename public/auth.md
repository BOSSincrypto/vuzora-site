# auth.md

## Граница аутентификации

Vuzora — статический сайт и неофициальный Telegram-сервис доставки расписания.
У Vuzora **нет регистрации и входа** на сайте, и нет пользовательской
аутентификационной сессии.

У опубликованного сайта нет OAuth/OIDC issuer, нет token endpoint или другого
endpoint для выдачи токенов, нет защищённого HTTP API и нет
аутентифицированного сервиса. Поэтому сайт не выдаёт токены, ключи или инструкции
для входа в несуществующую систему.

Переход в Telegram не подтверждает зачисление в вуз и не означает вход в
университетскую систему. Страница не является официальным сервисом вуза.

## Локальный WebMCP

Если браузер сам предоставляет экспериментальный `document.modelContext`,
страница может зарегистрировать только локальные read-only инструменты для
данных, уже включённых в сайт. Это браузерное улучшение, а не удалённый
MCP-сервер, не MCP Server Card, не HTTP API и не способ аутентификации. При
отсутствии этой возможности обычная навигация сайта не меняется.
